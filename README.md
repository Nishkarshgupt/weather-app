# 🌦️ Weather App

A simple **Weather Application** built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the **OpenWeatherMap API**.

Users can enter a **city name** and get the current weather details like **temperature, humidity, wind speed, and min/max temperature**.

---

## 🔗 Live Demo

👉 https://cloudmetric.netlify.app/

---

## 🚀 Features

* 🌍 Search weather by **city name**
* 🌡️ Shows **current temperature**
* 💧 Displays **humidity**
* 🌬️ Shows **wind speed**
* 📉 **Minimum temperature**
* 📈 **Maximum temperature**
* 🌤️ Weather condition icons
* ❌ Error handling for invalid city names
* ⌨️ Press **Enter** or click **Search Button**

## ⚙️ Setup Instructions

1️⃣ Clone the repository

```
git clone https://github.com/your-username/weather-app.git
```

2️⃣ Open the project folder

```
cd weather-app
```

3️⃣ Add your **OpenWeatherMap API Key** in `script.js`

```
const apiKey = "YOUR_API_KEY"
```

4️⃣ Open `index.html` in your browser.
